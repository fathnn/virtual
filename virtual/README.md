# VoiceConnect - Website Aplikasi Video Call

Website modern dan minimalis untuk aplikasi video call dengan desain astetik yang menarik.

![VoiceConnect](https://img.shields.io/badge/VoiceConnect-1.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 🎨 Fitur Desain

- **Minimalis & Astetik** - Desain clean dengan gradient yang elegan
- **Responsive** - Sempurna di desktop, tablet, dan mobile
- **Modern UI/UX** - Mengikuti tren desain terkini
- **Smooth Animations** - Animasi yang halus dan menarik
- **Fast Performance** - Optimasi untuk kecepatan loading

## 📱 Halaman-Halaman Utama

### 1. **Hero Section**
- Judul dan subtitle yang menarik
- Call-to-action buttons (Masuk/Daftar)
- Statistik pengguna (animasi)
- Video preview placeholder

### 2. **Features Section**
- 6 fitur unggulan dengan icon
- Hover effects yang interaktif
- Grid layout responsif

### 3. **How It Works**
- 3 langkah mudah dalam gradient background
- Step numbers dengan styling modern

### 4. **Pricing Section**
- 3 paket harga (Gratis, Premium, Bisnis)
- Badge "Populer" untuk paket terpilih
- Daftar fitur dengan checkmarks
- Paket featured dengan scale effect

### 5. **CTA Section**
- Call-to-action dengan gradient background
- Mendorong users untuk segera daftar

### 6. **Footer**
- Links organisasi
- Social media links
- Privacy & Terms links

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur semantik
- **CSS3** - Layout dengan Grid & Flexbox, animations
- **JavaScript (Vanilla)** - Interaktivitas tanpa framework
- **Font Awesome** - Icon library
- **Google Fonts** - Typography modern
